# Hardware - Resonador-432

Esta carpeta contiene los esquemas electrónicos, diseño de PCB y lista de componentes del dispositivo.

## Contenido planificado

- `schematic.pdf` — esquema electrónico completo.
- `pcb_layout.pdf` — diseño de la placa de circuito impreso.
- `components_list.md` — lista de componentes con referencias.
- `enclosure_design/` — diseño de la carcasa (STL para impresión 3D).

| Componente | Modelo / Referencia | Proveedor |
|------------|---------------------|-----------|
| Microcontrolador | ESP32 | Espressif |
| Acelerómetro | MPU-6050 | InvenSense |
| Motor DC | 5V, 200 mA | Estándar |
| Bobina / piezoeléctrico | 4-8 ohmios | Estándar |
